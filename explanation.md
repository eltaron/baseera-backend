## 📖 عن المشروع

**بصيرة (Baseera)** هي منصة تعليمية ذكية تهدف إلى تحسين تجربة التعلم للأطفال من خلال تحليل سلوكهم وتقديم محتوى مخصص باستخدام تقنيات الذكاء الاصطناعي.

المشروع لا يقتصر على عرض محتوى تعليمي فقط، بل يعمل على:

- فهم طريقة تعلم كل طفل 🧠
- تحليل تفاعله مع المحتوى 📊
- تقديم توصيات مخصصة 📈

## 📥 تثبيت المشروع

### 1. استنساخ المشروع

```bash
git clone https://github.com/eltaron/baseera-backend.git
cd baseera-backend
```

### 2. تثبيت الحزم

```bash
composer install
```

### 3. إعداد ملف البيئة

```bash
cp .env.example .env
```

### 4. إنشاء مفتاح التطبيق

```bash
php artisan key:generate
```

### 5. إعداد قاعدة البيانات

عدل بيانات الاتصال داخل `.env` ثم نفذ:

```bash
php artisan migrate
```

### (اختياري) إضافة بيانات تجريبية

```bash
php artisan db:seed
```

### 6. تشغيل المشروع

```bash
php artisan serve
```

## 🧠 مفاهيم لارافيل المستخدمة

### 1. Migration 📐

تصميم هيكل قاعدة البيانات بالكود بدلًا من العمل اليدوي.

### 2. Model 🧠

ربط قاعدة البيانات بالكود بطريقة سهلة ومنظمة.

### 3. Factory 🏭

توليد بيانات وهمية لاختبار النظام بسرعة.
